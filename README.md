# Customer Shopping Behavior Analysis Using Python, PostgreSQL, and Power BI

## 📊 Project Overview

This project analyzes customer shopping behavior to identify purchasing patterns, customer preferences, and key revenue drivers. The analysis was performed using Python for data cleaning and preprocessing, PostgreSQL for data storage and querying, and Power BI for creating interactive dashboards and business reports.

The dataset contains customer demographic information, purchase history, product categories, subscription status, discounts, payment methods, and shopping frequency. The project transforms raw customer data into actionable business insights that support data-driven decision-making.

---

## 🎯 Business Objectives

* Analyze customer purchasing behavior and spending patterns.
* Identify top-performing product categories.
* Understand customer demographics and shopping preferences.
* Evaluate the impact of discounts and promotional campaigns.
* Measure subscription program effectiveness.
* Build interactive dashboards for business reporting.
